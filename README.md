# Quiz Competition Application

A Java Swing-based quiz competition platform for Players to test their knowledge and compete.

## Features

- **Secure Login**: Access your account
- **User Registration**: Sign up for a new account
- **Take Quizzes**: Answer questions based on difficulty level
- **View Scores**: Track your quiz performance
- **Leaderboard**: See rankings of all players
- **Profile Management**: View your player details and scores

## Technology Stack

- **Language**: Java
- **GUI Framework**: Java Swing
- **Database**: MySQL
- **JDBC Driver**: MySQL Connector/J
- **Architecture**: MVC Pattern

## Setup Instructions

1. **Database Setup**:
   - Create database `CompetitionDB`
   - Run the SQL setup script to create tables

2. **Run Application**:
   - Run `GUI/QuizMainFrame.java` to start the application (Main.java is a placeholder).
